# üéâ Liva v1.0.0 - First Stable Release

Welcome to the first stable release of the Liva programming language!

## What is Liva?

Liva is a modern programming language that compiles to Rust. It combines:

- **TypeScript's clean syntax** - Familiar and readable
- **Python's expressiveness** - Get things done quickly  
- **Rust's safety and performance** - Memory-safe, zero-cost abstractions

## Quick Start

### 1. Install

```bash
git clone https://github.com/liva-lang/livac.git
cd livac
cargo build --release
cargo install --path .
```

### 2. Hello World

Create `hello.liva`:

```liva
main() => print("Hello, Liva!")
```

Run it:

```bash
livac hello.liva --run
```

### 3. Explore More

```liva
// Variables

let age = 1
const VERSION = "1.0.0"
```

## Example: GitHub API Client

```liva
import { HttpClient } from "http"
import { Json } from "json"

interface GitHubUser {
  login: string
  name: string
  followers: number
  public_repos: number
}

main() {
  let client = HttpClient.new()
  let response = client.get("https://api.github.com/users/torvalds").send()
  
  if response.ok {
    let user: GitHubUser = Json.parse_as(response.body)
    print($"üë§ {user.name} (@{user.login})")
    print($"üìä {user.followers} followers, {user.public_repos} repos")
  } else {
    print($"Error: {response.status}")
  }
}
```

## Example: Generic Collections

```liva
class Stack<T> {
  items: T[]
  
  new() {
    this.items = []
  }
  
  push(item: T) {
    this.items.push(item)
  }
  
  pop(): T {
    return this.items.pop()
  }
  
  isEmpty(): bool => this.items.length == 0
}

main() {
  let stack = Stack<number>.new()
  stack.push(1)
  stack.push(2)
  stack.push(3)
  
  while !stack.isEmpty() {
    print($"Popped: {stack.pop()}")
  }
}
```
