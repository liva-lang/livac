Implementa un servidor de chat TCP multi-cliente en Liva (.liva) usando rust { } interop con tokio (crate interno). Los usuarios pueden conectarse, elegir un nickname, enviar mensajes a todos (broadcast) o a un usuario específico con /msg nombre mensaje (mensaje directo). El servidor debe manejar desconexiones correctamente y mantener una lista de usuarios conectados. Guarda este prompt exacto en un archivo original_prompt.md dentro del proyecto.
